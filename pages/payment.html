怎麼運作

業務在後台建好收款單 → 系統自動產生帶 token 的連結 → 用 LINE 傳給房東或租客 → 對方手機打開頁面、確認金額後手寫簽名 → 系統即時產生 PDF 收據並回寫 Ragic，業務不用帶紙本合約出門。

運作中

2 帳戶同時比對
即時 LINE 查帳

怎麼運作

財務在 LINE 群組輸入 /查帳 → 機器人自動登入銀行後台 → 抓近期交易明細 → 比對租售帳戶和裝潢帳戶的入帳紀錄 → 回覆哪筆款項已入帳、哪筆還未到。不用財務自己開電腦登銀行每次手動查。
