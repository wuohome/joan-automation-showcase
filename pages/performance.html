怎麼運作

業績資料源頭在公司 Google Sheet → GAS 每日定時把資料鏡像到 Service Account → Python 腳本把結構化資料轉成 Markdown 格式寫進 vault → GitHub Actions 自動 push 更新 wuohome/ragic → staff-dashboard.html 前端即時呈現最新數字。整條鏈每天 17:30 自動跑完，Joan 不需要手動匯出任何東西。
